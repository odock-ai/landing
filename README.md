## What Odock.ai Delivers

- Unified interface across OpenAI, Anthropic, Groq, Bedrock, Vertex, Fireworks, vLLM, LM Studio, and MCP servers
- Virtual API keys with per-team quotas, rate limits, and audit trails
- Built-in AI firewall: prompt injection defense, jailbreak filtering, and data leakage protection
- Real-time budgets, token-level spend tracking, and adaptive routing/failover
- Extensible plugin engine for validation, transformations, and custom workflows
